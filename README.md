Managing the salary acquittance process in educational institutions is often a tedious, manual task involving spreadsheets, fragmented approvals, and paper-based workflows. These methods are prone to errors, lack transparency, and offer limited accessibility for faculty and administrative staff.

**SalQ** is a comprehensive, web-based application tailored specifically for educational institutions. It automates the entire salary lifecycle â€” from onboarding faculty and configuring salary structures to computing monthly payments and generating digitally signed salary slips.

Key features include:
- Role-based interfaces for **Admin**, **HR**, and **Faculty**.
- Automated salary processing via scheduled background jobs.
- CSV-based bulk uploads.
- Dynamic salary reports with filters and custom date range support.
- Secure authentication and role-based access control.
- Email notifications, audit logs, and digital signature support.
- Responsive UI with accessibility options like font size control.
